SOUNDWAVE 🔊
Engineered for Those Who Hear More

A premium audio equipment e-commerce website for amplifiers, speakers, cables, and custom sound systems.

 Features
- Browse premium audio products — Amplifiers, Speakers, Cables, Sound Systems
- Build Your Sound — custom amplifier configuration tool
- Next Wave — upcoming product showcase with countdown timers and notify me
- User authentication — Phone OTP, Facebook, Email (via Firebase)
- User dashboard — orders, cart, saved builds, account settings
- WAVE AI chatbot — powered by Groq, audio expert assistant
- Fully responsive — mobile and desktop

 Tech Stack
- Frontend: React + Vite
- Styling: Tailwind CSS / Custom CSS
- Authentication: Firebase Auth
- Database: Firebase Firestore
- AI Chatbot: Groq API
- Hosting: (coming soon)

 Getting Started

1. Clone the repository
git clone https://github.com/YOUR_USERNAME/SoundVery.git
cd SoundVery

2. Install dependencies
npm install

3. Set up environment variables
Copy .env.example to .env and fill in your keys:
cp .env.example .env

 4. Run locally
npm run dev

 Environment Variables
See .env.example for required variables.
Never commit your real .env file.

 Project Structure
src/
  components/     — reusable UI components
  pages/          — all page components
  firebase/       — Firebase config and auth logic
  assets/         — images and static files
